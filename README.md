# next-sets
> Set a lot value value

## install:
```bash
npm install -S afeiship/next-sets --registry=https://registry.npm.taobao.org
```

## usage:
```js
import 'next-sets';
const obj = {};

nx.sets(obj, {
  testA: 'A1',
  testB: 'B1',
  testC: 'C1',
  testD: 'D1',
  'path.key1.a.b': 'b-value1'
});
```
